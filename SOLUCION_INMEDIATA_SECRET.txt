  SOLUCIÓN INMEDIATA: Agregar NEXTAUTH_SECRET en Render

ERROR ACTUAL:
[next-auth][error][NO_SECRET] Please define a 'secret' in production.

SOLUCIÓN (3 minutos):

1. GENERA EL SECRET:
   - Ve a: https://generate-secret.vercel.app/32
   - Copia el resultado completo

2. VE A RENDER DASHBOARD:
   - https://dashboard.render.com
   - Tu Servicio → Settings → Environment

3. AGREGA LA VARIABLE:
   - Click en "Add Environment Variable"
   - Key: NEXTAUTH_SECRET
   - Value: (pega el secret que copiaste)
   - Save

4. HAZ DEPLOY:
   - Events → Manual Deploy → Deploy latest commit

═══════════════════════════════════════════════════════════════

IMPORTANTE:
- El nombre debe ser EXACTAMENTE: NEXTAUTH_SECRET
- El valor NO debe tener espacios al inicio o final
- El valor NO debe tener comillas
- Debes hacer click en "Save" después de agregarla
- Debes hacer un nuevo deploy después de guardarla
